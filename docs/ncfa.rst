Neuro-Causal Factor Analysis
============================

All code for reproducing the synthetic and real data results from our paper on `Neuro-Causal Factor Analysis <https://arxiv.org/abs/2305.19802>`_ can currently be found in this `repo branch <https://gitlab.com/alex-markham/medil/-/tree/develop>`_, which is in the process of being merged back into the main ``MeDIL`` package.
